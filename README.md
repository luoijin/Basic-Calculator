# Basic Calculator

A modern, visually appealing calculator website with video background and interactive tilt effects. Created in 2023, uploaded in 2025.

## 📋 Overview

This is a feature-rich calculator web application built with HTML, CSS, and JavaScript. It includes a cinematic video background and smooth tilt effects for an enhanced user experience.

## ✨ Features

- **Basic Calculator Functions**: Addition, subtraction, multiplication, division
- **Video Background**: Dynamic MP4 background for visual appeal
- **Tilt Effects**: Interactive card tilt using vanilla-tilt.js library
- **Responsive Design**: Works on desktop and mobile devices
- **Clean Interface**: Modern UI with smooth animations

## 💻 How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/luoijin/Basic-Calculator.git

2. Navigate to the Project Folder:
    cd Basic-Calculator

3. Open index.html on your web browser:
    open index.html  # On macOS
    # OR
    start index.html # On Windows
    # OR simply double-click the file

__Customization__
Styling: Modify assets/css/style.css to change colors, layout, and animations
Calculator Logic: Update assets/js/app.js to add new functions
Background: Replace assets/vid/bg.mp4 with your own video
Logo: Change assets/img/logo.png to your preferred image
